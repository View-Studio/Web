
< 배열 문자열 관련 메서드 >
===============================================================================

[ toString ]
-----------------------------------------------------------------------------------------------------------
 - 배열을 문자열로 바꾸어 반환한다  -

~ 문법 ~
array.toString() 

~ 특징 ~
(1). 배열의 값들을 ','로 연결한 문자열을 리턴합니다.

ex)
var arr =[ 1, 2, 3, 4 ];
console.log( arr.toString() );    // 1, 2, 3, 4
-----------------------------------------------------------------------------------------------------------



[ join ]
-----------------------------------------------------------------------------------------------------------
 - 배열의 모든 값들을 연결한 문자열을 리턴합니다.  -

~ 문법 ~
array.join(구분자) 
array.join(separator) 

~ 특징 ~
(1). 각각의 값들 사이에는 파라미터로 입력된 구분자(separator)가 들어가게 된다.
(2). separator를 입력하지 않은 경우, default로 ','가 들어갑니다.


ex)
var arr =[ 1, 2, 3, 4 ];
console.log( arr.join() );      // 1,2,3,4
console.log( arr.join( '-' ) );    // 1-2-3-4
console.log( arr.join('') );    // 1234
-----------------------------------------------------------------------------------------------------------
